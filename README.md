# module-3-assignment
This assignment covers the fundamentals of JavaScript, including variables, data types, operators, conditionals, and loops. It focuses on building a strong understanding of how to store data, perform operations, make decisions, and repeat tasks efficiently in programming.


## JavaScript Fundamentals Practice

#### Variables
Variables are used to store data in JavaScript. They allow us to save and reuse values throughout a program using keywords like `let`, `const`, and `var`.

#### Data Types
Data types define the kind of value stored in a variable. Common JavaScript data types include String, Number, Boolean, Null, and Undefined.

#### Operators
Operators are used to perform operations on values and variables. Examples include arithmetic operators (`+`, `-`, `*`, `/`), comparison operators, and logical operators.

#### Conditionals
Conditional statements help make decisions in a program. Using `if`, `else if`, and `else`, different blocks of code can run based on specific conditions.

#### Loops
Loops are used to execute a block of code repeatedly. JavaScript provides loops such as `for` and `while` to automate repetitive tasks efficiently.
